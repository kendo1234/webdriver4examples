# Webdriver 4 Demo

## Description

Just a quick tutorial-level set of tests to brush up on latest concepts of Selenium Webdriver

## Getting Started

### Dependencies

* Java 11
* Junit 5
* Webdriver Manager
* Selenium 4

### Installing

* Basic Maven project
```
mvn install
```

### Executing program

* Run from within IDE
* Run in CLI
```
mvn test
```
